A.4. Knoppix

Knoppix ディストリビューションはほとんど紹介する必要はないでしょう。Knoppix は最初の人気ディストリビューションで、ライブ CD を提供していました; これは言い換えれば、ハードディスク不要ですぐに使える Linux システムを実行する起動可能な CD-ROM です。マシンにインストール済みのいかなるシステムにも影響を与えません。Knoppix ディストリビューションは、利用可能なデバイスの自動検出機能を使うことで、ほとんどのハードウェア構成で動きます。CD-ROM にはおよそ 2 GB の (圧縮された) ソフトウェアが含まれ、DVD-ROM 版の場合収録されているソフトウェアのサイズはより大きくなります。
この CD-ROM を USB メモリに組み込むことで、ファイルと一緒にシステムを持ち歩いて、コンピュータに痕跡を残さずに仕事をすることが可能になります。Knoppix ディストリビューションはハードディスクを全く使わないことを覚えておいてください。Knoppix はデフォルトで LXDE (軽量グラフィカルデスクトップ) を採用していますが、DVD 版では GNOME と KDE も収録しています。また多くの他のディストリビューションではデスクトップとソフトウェアの他の組み合わせが提供されています。ライブ CD の作成を比較的簡単に行うことを可能にする live-build Debian パッケージのお陰で、ある程度これを実現することが可能になりました。
→ http://live.debian.net/
Knoppix はインストーラを備えている点に注意してください: Knoppix ディストリビューションをライブ CD で試した後に、性能を向上させる目的で Knoppix をハードディスクにインストールすることも可能です。
→ http://www.knopper.net/knoppix/index-en.html
